This Project

This project is a simple example of how to use Python, Postgresql with Timescaledb, Flask & Docker to create a simple data pipeline.

It was developed in my Macbook using PyCharm IDE.

How the pipeline works?

The data is collected from Binance API, we used historical data (last 30 days of 5min candles) and stored in a Postgresql database in a table called crypto_hist_ht indexed with Timescaledb .

I also used websocket to get data from Binance in realtime and stored in a table called crypto_ht.

With the data stored in Postgresql, we could use advanced queries with timescaledb unique features,

The Flask dashboard can be accessed at http://127.0.0.1:5029/

What could be improved for scaling?

The data pipeline could be improved by using Kafka for handling the realtime data.

Kafka could sink the data into Apache spark.

Apache Spark could persist the data into PostgreSQL with TimescaleDB extension.

With our data persisted, we could then create the dashboards using Grafana, Superset or other tools .
